## Broadcast Features

### Admin Commands:
- `/broadcast <message>` - Send text broadcast to all users
- `/broadcast` (reply to any message) - Broadcast the replied message (supports all media types)
- `/stats` - View bot statistics
- `/users [page]` - List all users with pagination
- `/health` - Check bot health

### Supported Media Types for Broadcast:
- ✅ **Text messages** (with Markdown formatting)
- ✅ **Photos/Images** (with captions)
- ✅ **Videos** (with captions)
- ✅ **Documents/Files** (with captions)
- ✅ **Audio files** (with captions)
- ✅ **Voice messages**
- ✅ **Stickers**
- ✅ **GIFs/Animations**
- ✅ **Polls**
